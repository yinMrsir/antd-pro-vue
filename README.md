# antd-pro-vue

基于ant-design-vue3封装的组件库，具有更多拿到即用的组件，快速搭建页面。

demo：https://gitee.com/yinMrsir/fool-admin-template

## 安装下载

- 下载地址：https://github.com/yinMrsir/antd-pro-vue.git

## 快速使用

```
# 安装依赖(建议使用yarn)

npm install
or
yarn install

# 启动

npm run dev
or
yarn dev
```
